# BlazorJQueryUIWrapper
BlazorJQueryUI is a Blazor Library that wraps jQueryUI in C# so you can use jQueryUI components and function without leaving the comfort of C#.

## Change log
- **Version 0.0.1 - 2020-07-29**
  - Functionality added
    - Project setup
